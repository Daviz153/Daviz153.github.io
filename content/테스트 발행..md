## 주요 개념

- **LM 나무위키**: Andrej Karpathy가 공개한 LLM 기반 개인 지식 위키 시스템. 로우 데이터 → AI 위키 생성 → 상호 참조의 3층 구조
- **복리 지식**: 자료를 추가할수록 기존 페이지가 풍부해지고 교차 참조가 늘어나는 지식 축적 방식
- **CLAUDE.md (스키마)**: 위키 운영 규칙·프로세스를 정의하는 파일. Claude Code가 이를 읽고 작업 방식을 결정
- **상호 참조 (Cross-link)**: 위키 페이지 간 연결 링크. AI가 자동으로 관련 개념을 연결
- **건강검진 모드 (Lint)**: 위키 전체를 점검해 모순·노후 정보·누락 링크를 탐지·수정하는 주기적 점검
- **Obsidian Web Clipper**: 웹 기사를 클릭 한 번으로 Obsidian raw 폴더에 저장하는 브라우저 확장
